# E-commerce-Sales-Customer-Insights-Analysis-using-SQL
Analyzed E-commerce data using SQL Server to uncover insights on customer behavior, sales trends, and product performance. Performed joins, aggregations, and window functions to identify top customers and revenue drivers. Built key metrics like total revenue and average order value, enabling data-driven decisions and improved business performance.
🛒 E-commerce Sales & Customer Insights Analysis
📌 Overview

This project explores an E-commerce dataset using SQL Server to uncover valuable insights into customer behavior, product performance, and sales trends. By working with multiple relational tables, the analysis simulates real-world business scenarios and supports data-driven decision-making.

🎯 Objectives
Understand customer purchasing behavior
Identify top revenue-generating customers
Analyze product performance and demand
Track sales trends over time
Evaluate order status and operational efficiency

🗂️ Dataset
customer – customer details
orders – order information and status
order_items – product-level order details
products – product information and pricing

⚙️ Tools & Technologies
SQL Server
SQL (Joins, Aggregations, Window Functions, CTEs)
📊 Key Analysis

👥 Customer Analysis
Identified high-value and repeat customers
Analyzed customer contribution to total revenue

📦 Product Analysis
Determined best-selling and highest revenue products
Performed category-wise performance analysis

💰 Sales Analysis
Calculated total revenue and average order value
Analyzed daily and monthly sales trends





🚚 Order Analysis
Evaluated Delivered, Cancelled, and Pending orders
Assessed operational efficiency
🚀 Advanced SQL
Applied RANK() to identify top performers
Used window functions for running totals
Detected inactive customers using LEFT JOIN
💡 Key Insights
A small segment of customers drives a large portion of revenue
Certain products consistently outperform others
Sales trends highlight peak demand periods
Order status analysis reveals operational gaps
Identified inactive customers for targeted engagement
🧠 Conclusion

This project demonstrates strong SQL skills and the ability to transform raw data into actionable insights. The analysis can support business decisions in marketing, customer retention, inventory planning, and revenue optimization.
